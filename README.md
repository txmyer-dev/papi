# Project Atlas - Public AI (PAPI)

A non-profit initiative to democratize access to large language models by aggregating free inference tiers into a single, reliable, OpenAI-compatible API.

## Project Structure

- `pitch/` - Pitch documents, slides, and one-pagers for grants and sponsors.
- `web/` - Source code for the primary landing page and web dashboard.
- `docs/` - Technical documentation, architecture diagrams, and design guides.
- `assets/` - Shared logos, images, and media files.
- `scripts/` - Automation scripts for deployment, testing, and data collection.
