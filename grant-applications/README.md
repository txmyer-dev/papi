# PAPI Grant Applications
This directory contains custom grant applications. Each submission is generated from the Master Grant One‑Pager (`../docs/MASTER-GRANT-ONE-PAGER.md`) and tailored to a specific funder.

## Structure
- Each application lives in its own subfolder (`grant-applications/<funder-name>/`).
- Every folder contains the full application document, plus any supporting materials (budgets, letters of inquiry, proof of concept).
